# [水平线](https://daringfireball.net/projects/markdown/syntax#hr)

要创建分割线，请在单独一行上使用三个或多个星号(`***`)、破折号(`---`)或下划线(`___`),并且不能包含其他内容。

``` markdown
***

---

___
```

***

---

___
