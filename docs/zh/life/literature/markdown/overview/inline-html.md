# [内联HTML](https://daringfireball.net/projects/markdown/syntax#html)

允许你在Markdown中嵌入Html语法标记。

```markdown
This is a regular paragraph.

<table>
    <tr>
        <td>Foo</td>
    </tr>
</table>

This is another regular paragraph.
```
