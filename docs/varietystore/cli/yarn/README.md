# 说明

## 安装 yarn

``` shell
$> npm install -g yarn
```

## 升级 yarn

``` shell
$> yarn set version latest
```

## 常用命令

### yarn install

安装项目依赖

- 示例

``` shell
$> yarn install
```

- 可选项

|定义|描述|
|:-|:-|
|-D,--dev|添加一个包作为开发依赖|

### yarn add

从项目中添加依赖项

- 示例

``` shell
$> yarn add vue@3.2.21
```

### yarn remove

从项目中移除依赖项

- 示例

``` shell
$> yarn remove vue
```

### yarn up

升级整个项目的依赖项

- 示例

``` shell
$> yarn upgrade vue@3.2.21
```
